## 一、惯性测量单元 (IMU)

### 1.1 PA-IMU-01D / PA-IMU-01G 惯性测量单元

**英文名称：** Inertial Measurement Unit

**产品特性：**
- 高精度 MEMS 陀螺仪和加速度计
- 体积小、重量轻
- 功耗低
- 可靠性高
- 抗冲击振动能力强
- 支持多种通信接口

**技术参数：**

| 参数 | 陀螺仪 | 加速度计 |
|------|--------|---------|
| 测量范围 | ±300°/s | ±10g |
| 零偏稳定性 | <0.5°/h | <1mg |
| 标度因数非线性 | <100ppm | <500ppm |
| 带宽 | >100Hz | >100Hz |

**应用领域：** 航空飞行控制、无人机导航、姿态测量

---

### 1.2 PA-IMU-02D03D 双轴光纤陀螺仪

**英文名称：** Dual-axis Fiber Optic Gyroscope

**产品特性：**
- 采用光纤陀螺技术
- 高精度、高可靠性
- 零偏稳定性优异
- 动态范围大
- 启动时间短
- 全固态设计，寿命长

**技术参数：**

| 参数 | 指标 |
|------|------|
| 测量范围 | ±75°/s / ±150°/s |
| 零偏稳定性 | <0.1°/h |
| 随机游走 | <0.005°/√h |
| 标度因数非线性 | <50ppm |
| 工作温度 | -40°C ~ +70°C |

**应用领域：** 航天器导航、船舶导航、高精度姿态测量

## 二、姿态航向参考系统 (AHRS)

### 2.1 PA-AHRS01 姿态航向参考系统

**英文名称：** Attitude and Heading Reference System

**产品特性：**
- 集成 GPS/GLONASS 接收机
- 三轴 MEMS 陀螺仪、加速度计和磁力计
- 实时输出姿态角、航向角
- 数据更新率高
- 支持 NMEA0183 协议
- 体积紧凑，安装方便

**技术参数：**

| 参数 | 指标 |
|------|------|
| 姿态精度（静态） | ±0.5° |
| 姿态精度（动态） | ±1.0° |
| 航向精度 | ±1.0° |
| 航向范围 | 0° ~ 360° |
| 数据更新率 | 100Hz |
| 供电电压 | 9~36V DC |
| 功耗 | <5W |

**应用领域：** 无人机飞行控制、船舶姿态监控、车辆稳定性控制

## 五、定位测姿系统 (GPS/INS)

### 5.1 PA-GS 定位测姿系统

**英文名称：** GPS Positioning and Attitude System

**产品特性：**
- 多频 GPS/GLONASS/北斗接收机
- 实时测姿定位
- 支持双天线/三天线配置
- 数据更新率高
- 支持 RTK 实时差分定位
- 兼容多种后处理软件

**技术参数：**

| 参数 | 指标 |
|------|------|
| 定位精度（单点） | <1.5m (CEP) |
| 定位精度（RTK） | <2cm |
| 测姿精度（航向） | ±0.2° (2m 基线) |
| 测姿精度（姿态） | ±0.5° |
| 数据更新率 | 20Hz / 100Hz |
| 通道数 | 584 |
| 跟踪信号 | GPS L1/L2, GLONASS L1/L2, BDS B1/B2 |

**应用领域：** 测绘、地理信息系统、移动测图
