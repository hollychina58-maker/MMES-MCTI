# 任务002：UI规划设计（参考苹果/WordPress科技模板）

## 任务描述
设计网站的整体UI风格规范，要求：
- 参考苹果官网/WordPress酷炫科技模板风格
- 页面包括：首页、产品列表页、多个产品详情页、联系页面
- 配色方案：科技感、专业、现代
